## Installation 📥

### Method 1: Direct Download
1. Download the `windows_optimizer-v3.0.bat` file
2. Right-click and select "Run as administrator"
3. Follow the on-screen menu options

### Method 2: From Releases
1. Visit [Releases](https://github.com/imagineSamurai/Windows-Optimizer/releases) or [Latest Version](https://github.com/imagineSamurai/Windows-Optimizer/releases/latest)
2. Download `windows_optimizer-v3.0.bat`
3. Run as administrator
4. Follow the on-screen menu options

## System Requirements ⚡
- Windows 10/11
- Administrator privileges
- Minimum 1GB RAM
- 2GB free disk space
- PowerShell 5.0 or later (for advanced features)

## Safety Features 🛡️
- Automatic backup option before changes
- System restore point creation
- Safe cleanup operations
- Clear operation feedback
- Confirmation prompts for critical operations
- No critical system file modifications

## Usage Guide 💻
1. Run as administrator
2. Choose from 39 optimization options
3. Follow on-screen instructions
4. Monitor progress indicators
5. Review operation results
6. Return to menu or exit
